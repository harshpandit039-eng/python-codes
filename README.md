# Python Practice Codes 

A collection of Python practice programs and exercises created to strengthen core programming concepts through hands-on learning and problem-solving.

## About

This repository contains various Python practice codes covering beginner to intermediate-level topics. The purpose of this repository is to improve coding skills, understand Python fundamentals, and build a strong foundation for software development and data-related roles.

##  Topics Covered

* Variables and Data Types
* Input and Output Operations
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`for`, `while`)
* Functions
* Recursion
* Lambda Functions
* String Manipulation
* List Operations
* Tuples, Sets, and Dictionaries
* List Comprehensions
* Object-Oriented Programming (OOP)
* Exception Handling
* Decorators
* Random Module
* Pattern Printing
* Basic Algorithms and Logic Building Exercises
